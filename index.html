DateTime Converter

FarStudio.Utility.Parser

Web Service serves to provide data management for clients (mobile and web) in the form of Json. Date confirmation is very important, because the requests and responses needed by clients are different, there are standard formats, utc etc. I made a DateTime Converter package in Web API 2 and MVC c# environment, which is simple and easy to use.

DateTime Converter Featuress

Quick

Because conversion in level Json, this make conversion not blocking or use many resource of memory in your business code.

Easy

Implement conversion is very simple and easy. Just implement in single place, all data write and read already implemented.

Customizable

You can custom format date time. Because so many different format, in fact different country has diferent format. And usually there are use UTC date fornat and not.

Default Value

We provide default value. If you didn't add anything format in this conversion, so we created deafault format.

gRPC MVC Client (.NET Core)

gRPCFarstMVC

gRPC is a modern open source high performance RPC framework that can run in any environment. It can efficiently connect services in and across data centers with pluggable support for load balancing, tracing, health checking and authentication. It is also applicable in last mile of distributed computing to connect devices, mobile applications and browsers to backend services(https://grpc.io/)

gRPCFarstMVC This is a client for gRPC services. Developed using MVC .NET Core, which is the .net core application is the latest and best technology from Microsoft today, which can be implemented not only on Windows but also on linux. So you can even publish on nginx, aws, docker, azure and IIS itself.

gRPC MVC Featuress

.NET Core

This is an awesome streamwork from Microsoft, where we can deploy on multi-platform.

MVC

mvc structure is one of the best, its web based SPA. Faster and more structured.

Protobuf

Protobuf is easier to bind to objects, comparing strings in JSON can be slow. As JSON is textual, its integers and floats can be slow to encode and decode because JON is not designed for numbers.

Pagination

gRPC-FarstMVC is equipped with powerful pagination where there is advanced development technology from X.PagedList.
